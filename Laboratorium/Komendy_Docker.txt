+-----------------+
| KOMENDY DOCKERA |
+-----------------+

Budowanie obrazu (kropka na końcu!): docker image build -t <nazwa>:<tag> .
				 np: docker image build -t client:latest .

Tworzenie i uruchomienie kontenera: docker run -d -p <lok.port>:<mapowany port> --name <nazwa_kontenera> <nazwa_obrazu>:<tag>
				np: docker run -d -p 5244:80 --name server-container server

Uruchomienia kontenera: docker start <nazwa_kontenera>
		    np: docker start server-container

Zatrzymanie kontenera: docker stop <nazwa_kontenera>
		   np: docker stop server-container

Wylistowanie obrazów: docker images

Wylistowanie uruchomionych kontenerów: docker container ps

Wypisanie logów danego kontenera: docker logs <nazwa_kontenera>
			      np: docker logs 303a1e1f3fbce87078ed1ea4a89c014888593d695d8e30e12a33f4da6c548c3d
